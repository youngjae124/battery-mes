## 구조 설명

### battery-mes-backend
기존 MES 핵심 비즈니스 로직을 담당하는 Spring Boot 서비스입니다.

### battery-mes-frontend
기존 운영 UI를 담당하는 React 서비스입니다.

### battery-mes-python
분석 전용 Python 마이크로서비스입니다.

- `routers/`: FastAPI 엔드포인트
- `services/`: 분석 로직
- `schemas/`: 요청/응답 모델
- `models/`: 향후 학습/예측 코드 자리
- `utils/`: 통계 계산 유틸
- `core/`: 설정 관리

## Java 백엔드 내 신규 확장 구조

- `config/PythonServiceProperties.java`
- `config/PythonAnalysisClientConfig.java`
- `client/analysis/*`
- `dto/analysis/*`
- `service/analysis/*`
- `controller/analysis/PythonAnalysisController.java`
