## 1. Rule-based SPC 분석

`POST /analysis/spc` — 샘플값 리스트를 받아 통계 지표를 계산합니다.

### 계산 항목
- 샘플 수, 평균, 최대·최소, 표준편차
- 공정 상태 판정: 표준편차 ≤ 0.2 → `NORMAL`, 초과 → `WARNING`
- USL/LSL 입력 시 Cp/Cpk 공정능력지수 계산

### Cp/Cpk 산식
- `Cp = (USL - LSL) / (6 * σ)`
- `Cpk = min((USL - μ) / (3 * σ), (μ - LSL) / (3 * σ))`
- σ는 모집단 표준편차(`ddof=0`) 사용 (R-bar/d2 기반 정식 Cpk의 근사치)
- `usl`/`lsl` 누락, `usl <= lsl`, `σ = 0` 시 `cp`/`cpk` → `null`

### 아키텍처
```
React → Spring Boot POST /api/analysis/report-summary
      → FastAPI POST /analysis/report-summary
```

```json
    "gradeCCount": 7

    "doneCount": 18,
```

## 4. VLM — 불량 이미지 분석

### 개요
이미지를 업로드하면 Gemini Vision이 불량 유형·시각적 특징·추정 원인·재발 방지 제안을 한국어로 분석합니다.
기존 `gemini-2.5-flash-lite` 모델에 이미지를 함께 전송하는 멀티모달 방식으로, **별도 모델 없이** 구현했습니다.

### 진입 방법

| 방식 | 설명 |
|------|------|
| **전체 페이지 FAB** | 로그인 후 모든 화면 오른쪽에 고정 표시되는 원형 버튼 클릭 → 이미지 분석 모달 오픈. 드래그로 위치 이동 가능. |
| **불량 등록 폼** | 품질관리 > 불량 탭 > 불량 등록 폼 내 이미지 업로드 → 분석 결과가 '설명' 필드에 자동 입력 |

### 아키텍처
```
React (FileReader → base64 data URI)
  → Spring Boot POST /api/analysis/defect-image (JSON body, Bearer JWT)
  → FastAPI POST /analysis/defect-image
  → LlmService.complete_with_image()
      → genai.Client.models.generate_content(
           model="gemini-2.5-flash-lite",
           contents=[ Part.from_bytes(image_bytes, mime_type), text_prompt ]
         )
  → 한국어 시각 분석 텍스트 반환
```

### 구현 완료 파일
- `battery-mes-python/app/services/defect_image_service.py` — 프롬프트 구성 + Gemini Vision 호출
- `battery-mes-python/app/schemas/analysis.py` — `DefectImageRequest` / `DefectImageResponse` 스키마
- `battery-mes-python/app/routers/analysis.py` — `/analysis/defect-image` 라우터
- `battery-mes-python/app/services/llm_service.py` — `complete_with_image()` 메서드
- `battery-mes-backend/.../dto/analysis/PythonDefectImageRequestDto.java`
- `battery-mes-backend/.../dto/analysis/PythonDefectImageResponseDto.java`
- `battery-mes-backend/.../client/analysis/PythonAnalysisClient.java`
- `battery-mes-backend/.../client/analysis/PythonAnalysisRestClient.java`
- `battery-mes-backend/.../controller/analysis/PythonAnalysisController.java`
- `battery-mes-frontend/src/components/common/AiImageModal.jsx` — 전체 페이지 공통 VLM 모달
- `battery-mes-frontend/src/lib/mesApi.js` — `fetchDefectImageApi()`
- `battery-mes-frontend/src/hooks/useQualityLogic.js` — 불량 폼 이미지 분석 핸들러
- `battery-mes-frontend/src/pages/QualityPage.jsx` — 불량 등록 폼 이미지 UI
- `battery-mes-frontend/src/App.jsx` — FAB 드래그 로직 + 모달 상태 관리

## Gemini API 설정

| 항목 | 값 |
|------|----|
| SDK | `google-genai` (구 `google-generativeai` deprecated) |
| 모델 | `gemini-2.5-flash-lite` |
| API 키 환경변수 | `GEMINI_API_KEY` (`.env` 파일, 절대 커밋 금지) |
| 모델 환경변수 | `GEMINI_MODEL` (`docker-compose.yml`에 명시) |
