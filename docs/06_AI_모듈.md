# 06. AI 모듈

## 현재 단계

현재 Python 서비스는 학습 모델 없이 Rule 기반 SPC 분석만 수행합니다.

## 현재 구현 범위

- 샘플값 리스트 입력
- 평균 계산
- 최대값 계산
- 최소값 계산
- 표준편차 계산
- 공정 상태 판정 (`NORMAL` / `WARNING`)

## 현재 판정 규칙

- 표준편차 `0.2` 이하: `NORMAL`
- 표준편차 `0.2` 초과: `WARNING`

## 미리 만들어 둔 확장 구조

### `app/models/train.py`
향후 RandomForest 또는 다른 모델 학습 코드를 넣을 자리입니다.

### `app/models/predict.py`
향후 예측 API 또는 배치 추론 코드를 넣을 자리입니다.

## 향후 추가 예정 기능

- RandomForest 기반 공정 이상 탐지
- 학습 데이터셋 생성 파이프라인
- joblib 모델 저장/로딩
- 모델 버전 관리
- Java ↔ Python 분석 결과 캐싱
