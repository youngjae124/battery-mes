# 06. AI 모듈

## 현재 단계

현재 Python 서비스는 학습 모델 없이 Rule 기반 SPC 분석만 수행합니다.

## 현재 구현 범위

- 샘플값 리스트 입력
- 평균 계산
- 최대값 계산
- 최소값 계산
- 표준편차 계산
- 공정 상태 판정 (`NORMAL` / `WARNING`)
- USL/LSL(규격 상하한) 입력 시 Cp/Cpk 공정능력지수 계산

## 현재 판정 규칙

- 표준편차 `0.2` 이하: `NORMAL`
- 표준편차 `0.2` 초과: `WARNING`

## Cp/Cpk 계산

`POST /analysis/spc` 요청에 `usl`, `lsl`을 함께 보내면 응답에 `cp`, `cpk`가 포함됩니다.

- `Cp = (USL - LSL) / (6 * σ)`
- `Cpk = min((USL - 평균) / (3 * σ), (평균 - LSL) / (3 * σ))`
- `usl`/`lsl`이 없거나, `usl <= lsl`이거나, 표준편차가 0이면 `cp`/`cpk`는 `null`
- `σ`는 기존 `standard_deviation`과 동일하게 모집단 표준편차(`ddof=0`)를 사용 — 서브그룹 내 변동만 보는 정식 Cpk 산식(R-bar/d2 기반)이 아닌 단순화된 근사치입니다.

## 미리 만들어 둔 확장 구조

### `app/models/train.py`
향후 RandomForest 또는 다른 모델 학습 코드를 넣을 자리입니다.

### `app/models/predict.py`
향후 예측 API 또는 배치 추론 코드를 넣을 자리입니다.
