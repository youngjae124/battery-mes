# 07. Battery MES 사용자 가이드

## 이차전지 제조 공정 개요

배터리 하나가 완성되기까지의 공정 순서입니다.

```
[전극 공정] → [조립 공정] → [화성·활성화] → [검사]
  믹싱기         조립 라인      화성 챔버       검사기
```

### 전극 공정 (Electrode)

양극·음극 소재를 만드는 단계입니다.

| 단계 | 내용 |
|---|---|
| 믹싱 | 양극재 분말 + 도전재 + 바인더를 용매에 섞어 슬러리(반죽) 제조 |
| 코팅 | 알루미늄(양극) / 구리(음극) 포일에 슬러리를 얇게 도포 |
| 건조·압연 | 건조 후 롤러로 눌러 두께 균일화 |
| 슬리팅 | 원하는 폭으로 절단 |

**소재 약어**

| 약어 | 의미 |
|---|---|
| NCM | 니켈-코발트-망간 양극재. 에너지 밀도 높음 → 전기차·노트북용 |
| LFP | 리튬-인산철 양극재. 안전하고 수명 길지만 에너지 밀도 낮음 → ESS·버스용 |
| 21700 | 배터리 폼팩터 (지름 21 mm, 길이 70 mm 원통형) |

### 조립 공정 (Assembly)

전극을 실제 배터리 형태로 조립하는 단계입니다.

| 단계 | 내용 |
|---|---|
| 권취·적층 | 양극 + 분리막 + 음극을 쌓거나 돌돌 말아 젤리롤 형성 |
| 삽입 | 원통형 캔에 젤리롤 삽입 |
| 전해액 주입 | 리튬 이온이 이동할 전해액 주입 |
| 밀봉 | 캡을 용접해 완전 밀폐 |

### 화성·활성화 공정 (Formation / Activation)

처음으로 충전·방전을 진행하는 단계입니다.

| 단계 | 내용 |
|---|---|
| 초충전 | 처음 전류를 흘려 내부 화학반응 시작 |
| SEI 층 형성 | 음극 표면에 보호막이 자연 생성 → 배터리 수명 결정 |
| 에이징(Aging) | 일정 온도·기간 방치 후 안정화 |
| 용량 선별 | 초기 용량 측정으로 제품 등급 결정 |

시스템 내 **에이징 상태(PENDING / PASS / FAIL)** 필드가 이 단계 결과를 나타냅니다.

### 검사 공정 (Inspection)

출하 전 최종 품질을 확인하는 단계입니다.

| 검사 항목 | 내용 |
|---|---|
| OCV 측정 | 무부하 상태 전압 측정 (정상: 3.6 ~ 4.2 V) |
| 용량 검사 | 실제 충전 용량이 규격 이상인지 확인 |
| IR 측정 | 내부 저항 측정 (낮을수록 좋음) |
| 외관 검사 | 누액·변형·찍힘 확인 |
| 등급 분류 | A / B / C 등급으로 분류 |

## 주요 용어 정리

### 검사 유형

| 코드 | 이름 | 시점 |
|---|---|---|
| IQC | Incoming Quality Control | 자재가 들어올 때 |
| IPQC | In-Process Quality Control | 공정 진행 중간 |
| OQC | Outgoing Quality Control | 출하 직전 최종 |

### 불량 유형

| 코드 | 이름 | 의미 |
|---|---|---|
| DELAM | Delamination | 전극 코팅이 포일에서 떨어지는 박리 불량 |
| SHORT | Internal Short | 양극·음극이 내부에서 접촉되는 단락 불량 (화재 위험) |
| LEAK | Electrolyte Leak | 케이스 밀봉 불량으로 전해액이 새는 불량 |
| OQCFAIL | Final Inspection Fail | 최종 검사 기준 미달 불합격 |

### SPC 통계 용어

| 용어 | 의미 |
|---|---|
| X-bar | 서브그룹 내 측정값들의 평균 |
| Range (R) | 서브그룹 내 최대값 − 최소값 |
| UCL / LCL | 관리 상한 / 하한선 — 이 범위 안이면 공정 안정 |
| USL / LSL | 규격 상한 / 하한선 — 고객이 요구하는 허용 범위 |
| Cp | 공정능력지수 (1.33 이상 우수) |
| Cpk | 공정 중심이 규격 중심에 가까운 정도 (1.0 이상 합격) |

## 기능 사용 가이드

### 권장 진행 순서

```
1. 로그인
2. 자재 등록 → BOM 등록
3. LOT 등록
4. 작업지시 등록 → 작업배정 등록
5. 설비 상태 변경
6. 검사 등록 (PASS / FAIL)
7. 불량 등록 (FAIL 건 기반)
8. SPC 데이터 등록
9. 보고서 조회
```

---

### 1. 로그인 / 회원가입

1. 접속 후 이메일·비밀번호 입력 → **로그인**
2. 신규 계정은 **회원가입** 탭에서 이름·이메일·비밀번호·역할 선택 후 등록
3. 로그인 성공 시 메인 대시보드로 이동

---

### 2. 메인 대시보드

로그인 직후 표시되는 화면입니다.

- **KPI 카드 6개**: LOT 현황, 작업지시 진행률, 생산 달성률, 설비 가동 현황, 검사 합격률, 품질 리스크
- **공정 스냅샷**: 전극 / 조립 / 화성 / 검사 공정의 DONE 작업지시 수
- **LOT 감시 목록**: 진행중 LOT 빠른 확인
- **설비 포커스**: 현재 DOWN / PM 설비 목록
- 왼쪽 메뉴에서 각 도메인으로 이동

---

### 3. 자재 / BOM 관리 (Material 메뉴)

**자재를 먼저 등록해야 BOM 구성이 가능합니다.**

#### 자재 등록

| 필드 | 예시 |
|---|---|
| 자재 코드 | MAT-NCM-001 |
| 자재명 | NCM Cathode Powder |
| 자재 유형 | RAW_MATERIAL |
| 재고 수량 | 500 |
| 단위 | kg |

#### BOM 등록

| 필드 | 예시 |
|---|---|
| 제품 코드 | CELL-21700-NCM |
| 자재 선택 | 드롭다운에서 선택 (단위 자동 입력) |
| 단위당 소요량 | 0.015 |

하단 목록에서 **수정** 버튼으로 수정 가능합니다.

---

### 4. 생산관리 (생산관리 메뉴)

LOT → 작업지시 → 작업배정 순서로 진행합니다.

#### LOT 등록

| 필드 | 예시 |
|---|---|
| LOT 번호 | LOT-20260702-001 |
| 제품명 | 21700 CELL - NCM |
| 수량 | 500 |
| 상태 | 진행중 (IN_PROGRESS) |

#### 작업지시 등록

| 필드 | 예시 |
|---|---|
| 작업지시 번호 | WO-EL-003 |
| LOT | 드롭다운에서 선택 |
| 설비 | EQ-MIX-001 선택 |
| 공정 | 전극 |
| 상태 | 계획 (PLANNED) |
| 목표 수량 | 500 |
| 실적 수량 | 0 |
| 계획 시작일 | 날짜·시간 선택 |

공정이 완료되면 상태를 **DONE**으로 변경하면 상단 공정 흐름 패널에 **완료** 배지가 표시됩니다.

#### 작업배정 등록

| 필드 | 예시 |
|---|---|
| 작업지시 | 드롭다운에서 선택 |
| 작업자 | 사용자 목록에서 선택 |
| 배정 역할 | OPERATOR |
| 시작 일시 | 날짜·시간 선택 |
| 종료 일시 | 선택 (완료 후 입력) |

---

### 5. 설비관리 (설비관리 메뉴)

#### 설비 상태 변경

1. 하단 **설비 현황** 목록에서 수정할 설비의 **수정** 버튼 클릭
2. 좌측 폼에 설비 정보 자동 입력됨
3. **상태** 변경: RUNNING / IDLE / DOWN / PM
4. **로그 유형** 선택: BREAKDOWN / ALERT / PM
5. **로그 설명** 입력 후 **설비 수정** 버튼 → 상태 변경과 로그가 동시에 저장됨

설비 상태 의미:

| 상태 | 의미 |
|---|---|
| RUNNING | 가동 중 |
| IDLE | 대기 중 (정상이지만 작업 없음) |
| DOWN | 고장 (생산 불가) |
| PM | 예방 정비 중 |

#### 공정 파라미터 기록

| 필드 | 예시 |
|---|---|
| 작업지시 | 드롭다운에서 선택 |
| 파라미터명 | Temperature |
| 목표값 | 85.0 |
| 실측값 | 85.3 |
| 단위 | degC |
| 상한값 | 90.0 |
| 하한값 | 80.0 |

---

### 6. 품질관리 (품질관리 메뉴)

**검사 → 불량** 순서로 진행합니다. FAIL 판정이 난 검사만 불량 등록 대상이 됩니다.

#### 검사 등록 (검사 관리 탭)

| 필드 | 예시 |
|---|---|
| LOT | 드롭다운에서 선택 |
| 작업지시 | 해당 LOT의 WO 선택 (없으면 선택 안 함) |
| 검사 유형 | IQC / IPQC / OQC |
| 검사항목 | Open circuit voltage |
| 최소 기준값 | 3.6 |
| 최대 기준값 | 4.2 |
| 측정값 | 3.98 |
| 에이징 상태 | PENDING / PASS / FAIL |
| 비고 | 선택 입력 |

> 측정값 입력 시 **예상 판정(PASS / FAIL)과 등급(A / B / C)이 실시간으로 미리보기** 됩니다.
> 최종 판정은 저장 시 백엔드에서 계산합니다.

- 검사 목록에서 **CSV 내보내기** 가능
- ADMIN 계정은 **삭제** 버튼이 추가로 표시됨 (소프트 삭제)

#### 불량 등록 (불량 관리 탭)

| 필드 | 예시 |
|---|---|
| 검사 이력 | FAIL 판정 검사만 드롭다운에 표시 |
| 불량 유형 | DELAM / SHORT / LEAK / OQCFAIL |
| 심각도 | CRITICAL / MAJOR / MINOR |
| 설명 | 전극 접착력 기준 미달 |

하단 목록에서 심각도 배지(빨강 / 주황 / 회색)로 빠르게 위험도 확인 가능합니다.

---

### 7. SPC (SPC 메뉴)

공정이 통계적으로 안정한지 확인하는 화면입니다.

#### SPC 데이터 등록

| 필드 | 예시 |
|---|---|
| LOT | 드롭다운에서 선택 |
| 작업지시 | 드롭다운에서 선택 |
| 파라미터명 | 전극 두께 |
| 서브그룹 번호 | 11 |
| 샘플값 목록 | 80.1, 79.9, 80.3, 80.0, 80.2 (쉼표·줄바꿈 구분) |
| UCL | 83.0 |
| CL | 80.0 |
| LCL | 77.0 |
| USL | 85.0 |
| LSL | 75.0 |

1. **Cp/Cpk 미리보기** 버튼 클릭 → Python FastAPI를 호출해 공정능력지수 계산
2. 결과 확인 후 **SPC 등록** 버튼

#### 조회 및 관리도

- 우측 조회 패널에서 LOT / 작업지시 / 파라미터명 / 상태(정상·이탈) 필터
- **관리도 조회** 버튼 → X-bar / Range 데이터를 서브그룹 순으로 확인
- UCL 또는 LCL 이탈 건은 **빨간 배지(상한 이탈 / 하한 이탈)** 로 표시

---

### 8. 보고서 (보고서 메뉴)

> ADMIN / OPERATOR / INSPECTOR 역할만 접근 가능합니다.

1. 조회 날짜 선택 (기본값: 오늘)
2. **조회** 버튼
3. 두 보고서가 동시에 표시됩니다

| 보고서 | 내용 |
|---|---|
| 일간 품질 보고서 | 해당 날짜의 검사 합격률, PASS / FAIL 건수, 불량 수, A등급 비율 |
| 생산 실적 보고서 | 공정별(전극 / 조립 / 화성 / 검사) 목표 대비 실적 달성률 |

### SPC
