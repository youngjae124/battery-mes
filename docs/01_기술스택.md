## 기술 선택 이유

### FastAPI
분석용 API를 빠르게 분리하고 Swagger(`/docs`)를 기본 제공하기 때문에, 기존 Java MES와 독립된 분석 마이크로서비스를 붙이기에 적합합니다.

### numpy / pandas
SPC 계산과 향후 데이터 전처리 로직을 확장하기 쉽습니다.

### scikit-learn / joblib
현재는 모델 학습을 하지 않지만, 이후 RandomForest 모델을 추가할 때 바로 이어서 확장할 수 있습니다.
